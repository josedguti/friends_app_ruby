# Ruby on Rails Friends List

Friends List App where you can Sign Up, Sign In, and then start adding your contacts to your personal list.

## ⚙️ Technologies Used

- Ruby
- Rails
- HTML5
- CSS/Bootstrap
- GitHub
- Heroku (for deployment)
